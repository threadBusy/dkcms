编号 接口名称 输入参数 返回数据 备注
1.1 获取栏目列表
url:/openapi/getCategoryTree		 基础参数:token,sign
业务参数:无 		标准返回结构
1.2 提交内容API
url:/openapi/postContent		 基础参数:token,sign
业务参数:title|content|categoryId 等文章内容需要的字段，但至少有这三个字段
业务参数:saveAsDraft=false 如果明确指定了这个参数为false，则文章会正式发布，否则默认保存入草稿箱
 标准返回结构 提交文章，进入草稿箱，需登录后台手动发布
1.3 重新生成首页接口
url:/openapi/generateSiteHomePage		 基础参数:token,sign
业务参数:无 		标准返回结构
1.4 重新生成栏目页面接口
url:/openapi/generateCategoryPage		 基础参数:token,sign
业务参数:categoryId(必填，合法栏目id，不能是 0 )
业务参数:withChild(选填，默认false,是否包含子栏目)
 标准返回结构
